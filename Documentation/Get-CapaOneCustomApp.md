# Get-CapaOneCustomApp

## SYNOPSIS
Retrieves custom applications from CapaOne.

## SYNTAX

```
Get-CapaOneCustomApp [[-Id] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Lists all custom applications or a specific app when an ID is supplied.

## EXAMPLES

### EXAMPLE 1
```
Get-CapaOneCustomApp
```
