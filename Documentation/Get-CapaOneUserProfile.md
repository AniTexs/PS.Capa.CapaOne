# Get-CapaOneUserProfile

## SYNOPSIS
Retrieves the current user's profile information.

## SYNTAX

```
Get-CapaOneUserProfile [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Requests the user profile for the connected organization.

## EXAMPLES

### EXAMPLE 1
```
Get-CapaOneUserProfile
```
