# New-CapaOneGroup

## SYNOPSIS
Creates a new CapaOne group.

## SYNTAX

```
New-CapaOneGroup [[-Name] <String>] [[-Description] <String>] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
Adds a group with the specified name and optional description.

## EXAMPLES

### EXAMPLE 1
```
New-CapaOneGroup -Name "HR"
```
